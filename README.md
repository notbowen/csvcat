# CSVCat

A blazingly fast (vibe-coded) `batcat`-like tool for viewing CSV files in the
terminal. Currently optimised for MacOS. I created this project to test the
capabilities of Claude Code and to look for ways to integrate it into my daily
coding workflow.

## Usage

Clone this repository and run `cargo build --release` or `cargo install --path
.`. Use an LLM to explain to you the detailed steps if needed.
